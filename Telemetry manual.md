# Telemetry Manual
Basic instructions on how to use the telemetry app.

You can access the app by typing in localhost:3000 into your browser URL.
If you want to have admin access you need to login with credentials with
instructions below.

## How to login
Type in localhost:3000/login into your browser URL and login with your credentials.

## How to use
You can access the drilldown to see metrics and logs

## How to logout
Type in localhost:3000/logout into your browser URL and you have logged out.
